# solitaire

山东大学软件学院面向对象实验 纸牌游戏 solitaire game.这个项目采用flutter框架制作，语言使用dart语言。对于flutter框架下面readme有介绍。已经release的版本在window10上是可以直接运行的。

## Getting Started 安装flutter
按照flutter官方教程安装flutter框架
https://flutter.dev/docs/get-started/install

## 运行平台代码
需要在项目文件夹下执行
1. windows平台: flutter build windows
2. 安卓平台: flutter build android
3. etc...
